# Fuels_france

Fuels_france allows you to observe the price of fuel on the French territory

## Installation

Install Docker.

Install docker-compose.

Start Docker.

Go in the folder Fuels_france/ :

```bash
cd Fuels_france/
```

Run :

```bash
docker-compose up -d
```

You can now use the App :

```bash
App running at:
  - Local:   http://localhost:8080/
```
